5. 서블릿(Servlet)

  1)특징
    - *.java
    - 저장위치는 src/main/java에 저장
    - 반드시 패키지 필수 (****)
    - main 메서드가 없음. 시작점이 없다.
     ( 이유는 사용자가 맨 처음 요청하는 서블릿이 시작점이 됨 )

   - 직접 new하지 않음.
     Tomcat 서버가 필요한 시점에 생성부터 소멸까지 관리해줌.
     즉 Tomcat 서버가 필요한 이유가 Servlet/JSP 같은 동적컴포넌트를 관리함.

    - 실행결과는 html로 나옴.

 2) 구현 방법 (외우기)

     가. 패키지로 구성
        ex. com.servlet

     나.  extends HttpServlet   (  jakarta.servlet.http 패키지 )

   라. 서블릿맵핑

       예>
           src
	     main
	        java
		  com
		    servlet
		       HelloServlet


        요청:  http://localhost:8080/컨텍스트명/servlet/풀패키지명의서블릿명

	      http://localhost:8080/hello/servlet/com.servlet.HelloServet <== 현재 사용 안됨


	      사용안한 이유 2가지
	      - 보안취약
	      - URL 길다.


	     해결: 서블릿 맵핑(servlet mapping)

	     http://localhost:8080/hello/servlet/com.servlet.HelloServet

	     http://localhost:8080/hello/서블릿맵핑

     
      * 서블릿 맵핑 방법 2가지

        가. 어노테이션 이용
	    - @WebServlet("/서블릿맵핑")  # / 필수


	나. web.xml 

              <!-- 서블릿맵핑 -->
	    <servlet>
		<servlet-name>WorldServlet</servlet-name>
		<servlet-class>com.servlet.WorldServlet</servlet-class>
	    </servlet>
	   <servlet-mapping>
		<servlet-name>WorldServlet</servlet-name>
		<url-pattern>/world</url-pattern>
	    </servlet-mapping>	    
 
	  ===> Spring Framework에서 내부적으로 사용되는 서블릿(DispatcherServlet, / 으로 매핑되어있음)을 
	      매핑하는 방법임.

6. 서블릿 LifeCycle

   - Tomcat 컨테이너가 Servlet의 LifeCycle(생성~소멸)관리함.
    생성 때 자동 호출되는 콜백메서드(시스템이 호출하는 메서드) 제공 : init()
    소멸 때 자동 호출되는 콜백메서드 (시스템이 호출하는 메서드) 제공 : destroy()
    요청 때 자동 호출되는 콜백메서드 (시스템이 호출하는 메서드) 제공 : doGet/doPost

   1) init() 메서드

      - 서블릿이 생성될 때 자동으로 호출됨.
      - 한번만 호출됨

   2) doGet/doPost 메서드
   
      - 서블릿에 요청할 때 자동으로 호출됨.
      - 요청할 때마다 매번 실행됨. (*********************************)

       코드구현:
             doGet(HttpServletRequest request, 
	           HttpServletResponse response){}

             doPost(HttpServletRequest request, 
	           HttpServletResponse response){}
