Servlet/JSP 강좌정리 2026-05-04

1. 자바 개발 플랫폼 3가지

  1) Java SE ( Java Standard Edition )
    - 자바 spec
    - JDK 설치 ( JVM, 개발툴(javac, java), API(.jar))
    - 시작점(starting poing)인 main 메서드를 가짐.

  2) Java EE ( Java Enterprise Edition )
    
    - 웹 어플리케이션 개발 플랫폼.
    - Servlet/JSP spec ( 인터페이스 )
	      
    - 실제 구현체를 제공하는 벤더는 Apache임.
      ( Tomcat 서버와 Servlet/JSP의 jar 제공)

  3) Java ME ( Java Micro Edition )

3. 웹프로젝트 생성

  File > new > Dynamic Web Project 선택 >
  
    Web Module 화면

       Context root:  논리적인 이름인 컨텍스트명, 나중에 변경가능,
                      기본적으로 프로젝트명으로 설정됨.
		      http://localhost:8080/컨텍스트명 (매우중요 ******************)

       Content Directory: src/main/webapp  <== 이클립스가 관리하는 디렉토리

        Generate Deployment Descriptor: web.xml (배치 지시자)
	                                하나의 웹 어플리케이션 설정정보 관리.
